# BERT-BASED-FAKE-NEWS-DETECTION-SYSTEM

A full-stack AI project that detects fake news using BERT.

## ▶️ RUN PROJECT

## BACKEND
cd backend
uvicorn app:app --reload

## FRONTEND
Open frontend/index.html

## 🚀 FEATURES
- Fake/Real News Detection
- Confidence Score
- Modern UI
- FastAPI Backend

## 🛠 TECH STACK
- Python, FastAPI
- Transformers (BERT)
- HTML, CSS, JavaScript
